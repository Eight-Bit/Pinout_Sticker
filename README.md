# Pinout Sticker Library

This is an libary for IC Pinout Stickers.  
Just print them on adhesive paper or use glue to fix them on your ics.

## List

1. ATmega16
2. ATmega32

## Format

All stickers are available as:

- PDF  
- JPEG
